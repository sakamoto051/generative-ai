# ProCost Manager MVP (Phase 1) Implementation Plan

## 概要
生産計画最適化・原価管理システム（ProCost Manager）のMVP実装計画

## 実装期間
3ヶ月（フェーズ1）

## MVP機能範囲

### 1. 基本マスタ管理
- ユーザー管理（認証・権限）
- 製品マスタ
- 材料マスタ
- BOM（部品表）マスタ
- 設備マスタ
- 作業者マスタ

### 2. 生産計画立案
- 需要データ入力
- 生産計画の手動作成
- 材料所要量計算（MRP）
- 計画の保存・編集

### 3. 製造指示・実績管理
- 製造指示書発行
- 材料払出記録
- 作業実績入力
- 進捗管理

### 4. 簡易原価計算
- 材料費計算
- 労務費計算
- 製造経費の簡易配賦
- 製品別原価集計

### 5. 基本レポート
- 生産計画一覧
- 製造実績レポート
- 原価計算結果レポート

## データベース設計

### 主要テーブル（MVP）

1. **users** - ユーザー
2. **roles** - ロール
3. **permissions** - 権限
4. **products** - 製品マスタ
5. **materials** - 材料マスタ
6. **boms** - BOMマスタ
7. **equipment** - 設備マスタ
8. **workers** - 作業者マスタ
9. **demands** - 需要データ
10. **production_plans** - 生産計画
11. **production_plan_items** - 生産計画明細
12. **manufacturing_orders** - 製造指示
13. **material_issues** - 材料払出
14. **work_results** - 作業実績
15. **costs** - 原価計算結果
